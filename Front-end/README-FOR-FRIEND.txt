This is a React Native mobile app project.

═══════════════════════════════════════════════════════════════
QUICK START (EASIEST WAY)
═══════════════════════════════════════════════════════════════

1. Copy this folder from pendrive to your laptop
   (Use a short path like: C:\Projects\vimmi404)

2. Double-click: INSTALL-AND-RUN.bat
   (This will install everything and run the app automatically)

That's it! The app will start on your Android device or emulator.

═══════════════════════════════════════════════════════════════
MANUAL SETUP (If batch file doesn't work)
═══════════════════════════════════════════════════════════════

1. Install Node.js from: https://nodejs.org/
   (Download and install the LTS version)

2. Open PowerShell in this folder

3. Run: npm install
   (Wait 5-10 minutes)

4. Connect Android phone OR start Android Studio emulator

5. Run: npx react-native run-android

═══════════════════════════════════════════════════════════════
REQUIREMENTS
═══════════════════════════════════════════════════════════════

- Windows 10/11
- Node.js (v18+) - Download from nodejs.org
- Android Studio (for Android development)
- Android phone with USB debugging OR Android Studio Emulator

═══════════════════════════════════════════════════════════════
IMPORTANT NOTES
═══════════════════════════════════════════════════════════════

- First build takes 5-10 minutes (this is normal)
- Use a SHORT folder path (avoid long paths)
- Make sure Android device/emulator is connected before running
- Internet connection required for first setup (npm install)
