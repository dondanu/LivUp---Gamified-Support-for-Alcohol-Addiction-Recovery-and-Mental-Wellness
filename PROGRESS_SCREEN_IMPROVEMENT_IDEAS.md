#### 1. **Weekly Summary Card** ⭐⭐⭐⭐⭐
**என்ன?**

- Shows improvement or decline
- Visual up/down arrows

**Example:**
```
┌─────────────────────────────────────┐
│  📊 This Week vs Last Week          │
├─────────────────────────────────────┤
```

## 📊 Technical Requirements

### New Backend APIs Needed:
1. **Weekly Comparison** - `GET /api/progress/weekly-comparison`
2. **Monthly Goals** - `POST/GET /api/goals`
3. **Best Day** - `GET /api/progress/best-day`
4. **Stats Summary** - `GET /api/progress/summary`

### New Database Tables:
1. **monthly_goals** - Store user goals
   ```sql
   CREATE TABLE monthly_goals (
     id INT PRIMARY KEY,
     user_id INT,
     month VARCHAR(7), -- YYYY-MM
     goal_type VARCHAR(50), -- 'sober_days', 'drink_limit'
     target_value INT,
     current_value INT,
     created_at TIMESTAMP
   );
   ```

### New Components:
1. `WeeklySummaryCard.tsx`
2. `StreakVisualization.tsx`
3. `QuickStatsCard.tsx`
4. `MonthlyGoalsCard.tsx`
5. `BestDayCard.tsx`
